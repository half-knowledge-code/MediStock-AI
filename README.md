📦 MediStock AI

🚑 AI Powered Pharmacy Inventory Management System

MediStock AI is an intelligent pharmacy inventory platform designed to help pharmacies manage medicine stock, monitor expiry dates, predict demand using machine learning, and ensure proper storage conditions with automated alerts.

🚀 Key Features
💊 Smart Medicine Inventory
Track medicines with batch, quantity, and expiry
Real-time inventory monitoring
⏳ FEFO Algorithm
First Expiry First Out
Ensures medicines with earliest expiry are sold first
⚠ Expiry Alerts
Detect medicines expiring within 60 days
📉 Low Stock Detection
Alerts when medicine quantity falls below threshold
🤖 AI Demand Prediction
Uses Machine Learning (Linear Regression)
Predicts future medicine demand
🌡 Sensor Monitoring
Tracks temperature and humidity
Ensures safe storage conditions
📧 Automatic Email Alerts
Sends email alerts if temperature exceeds safe limits
📊 Analytics Dashboard
Displays inventory statistics
Stock insights and reports
📄 PDF Inventory Reports
Export pharmacy inventory as PDF
